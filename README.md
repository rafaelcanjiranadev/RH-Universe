RH Universe

Estado do Projeto: Em desenvolvimento

Tecnologias que serão aplicadas: Java e MySql

Time de desenvolvedores: Rafael (Dev)

Objetivo do software: Objetivo do sistema é ajudar funcionários da Gestão de "Recursos Humanos" a automatizar 
e otimizar suas rotinas diárias de uma maneira simples e rapida de entender.

Funcionalidades do sistema: Cadastro de funcionários, permissões de acesso, tela de login, CRUD em todas as telas, 
filtros dinâmicos para buscar, cadastro e calculo de pagamentos, visualização a folha de pagamento dos funcionários.
